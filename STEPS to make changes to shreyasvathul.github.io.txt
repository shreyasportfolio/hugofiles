git clone https://github.com/shreyasportfolio/hugofiles.git .

git clone https://github.com/sethmacleod/dimension.git themes/dimension/

5. test out put "hugo server". Trasfer files from public folder to shreyasportfolio.github.io folder

6. Make changes and upload to github (separately for the project folder and the shreyasportfolio.github.io folder
